UFS MR Weather App Documentation

Here you can find a Users's Guide, links to external documentation, links to the development repository, and access to the UFS Portal

In the list below, you see 3 types of links: 1) from HTML stored in ufs-weather model-repository, branch gh-pages; 2) from HTML stored in github.io; 3) link to external documentation.

At this time, my recommendation is is to put the documentation close to the code, so put the HTML in ufs-weather model-repository, branch gh-pages. If we want to have different versions of documentation (e.g., release 0.1, 1.0, etc., we put the HTML in different subdirectories). Note that he branch must be called gh-pages (we cannot change this name). The other alternative is to put the HTML in ufs-weather-app master, subdirectory docs. But I think the gh-pages branch may be a better solution". At any rate, the source code for documentation will be in the ufs-weather-aqpp repo".
